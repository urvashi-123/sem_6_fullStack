# Node Class — Notes & Examples

This repo contains structured notes and runnable examples for UNIX, Git, Bash, and Node.js.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run examples:

```bash
npm run examples
```

3. Run tests:

```bash
npm Test
```
